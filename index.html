Lưu ý quan trọng

Nguồn phát video mà script đang sử dụng có thể chứa quảng cáo. Các quảng cáo này được nhúng trực tiếp vào file M3U8. Việc cố gắng bỏ qua có thể dẫn đến việc phải xem lại từ đầu.

Cài Đặt Dễ Dàng

Trên Ubuntu/Debian

Nếu bạn không install đưọc bằng lệnh dưói thì cài riêng từng package bị lỗi nhé

sudo apt install ffmpeg curl grep yt-dlp fzf pup manga-tui jq awk cut

Trên Arch Linux

yay -S ffmpeg curl grep yt-dlp fzf pup manga-tui jq awk cut

Khởi chạy & Sử dụng

Cấp quyền & Chạy Script

# Cấp quyền thực thi cho file
chmod +x anisub.sh

# Chạy file trực tiếp
./anisub.sh

Tạo Alias (Lối tắt)

Để tiện lợi hơn, hãy tạo một lối tắt trong file `.bashrc` hoặc `.zshrc` của bạn. Dòng lệnh này giúp bạn chạy script chỉ bằng một từ khóa ngắn gọn.

# Thay thế đường dẫn tới file anisub.sh
alias anisub='bash $HOME/path/to/anisub.sh'

Sau khi thêm, khởi động lại terminal và gõ `anisub` để bắt đầu. Chúc bạn thành công!
